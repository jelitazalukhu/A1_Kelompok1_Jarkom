# PERANCANGAN TOPOLOGI DASAR PADA JARINGAN PERUSAHAAN
Proyek ini merupakan simulasi topologi jaringan perkantoran yang dirancang menggunakan Cisco Packet Tracer. 
Topologi ini membagi jaringan ke dalam beberapa segmen berdasarkan fungsi dan ruangan, 
dengan layanan utama berupa DHCP, HTTP, DNS, routing statis, dan jaringan wireless.
